# 1.  제목 작성 - Headers

 - H1 ~ H6 까지 크기 지정이 가능하다
 - 앞에붙은 "#" 개수에 따라 바뀐다.
 - H1과 H2의 경우 각 ===== 와 -----로 대체 가능하다

```
# 헤더 H1
## 헤더 H2
### 헤더 H3
#### 헤더 H4
##### 헤더 H5
###### 헤더 H6
```

# 헤더 H1
## 헤더 H2
### 헤더 H3
#### 헤더 H4
##### 헤더 H5
###### 헤더 H6
```
헤더 H1
=====
```

헤더 H1
=====
```
헤더 H2
-----
```

헤더 H2
-----

# 2.  강조 - Emphasis
 - 언더바(_) 와 별(*) 를 이용하여 글씨를 강조할 수 있다.
 - html 문법을 통해서도 강조 할 수 있다.

```
이텔릭체 => *이텔릭체* / _이텔릭체_  
두껍게 => **두껍게** / __두껍게__  
두꺼운 이텔릭체 => **_두꺼운 이텔릭체_** / __*두꺼운 이텔릭체*__  

취소선 => ~~취소선~~
밑줄 => <u>밑줄</u>
```

이텔릭체 => *이텔릭체* / _이텔릭체_

두꺼운 이텔릭체 => **_두꺼운 이텔릭체_** / __*두꺼운 이텔릭체*__
