# 1.  제목 작성 - Headers  

 - `H1` ~ `H6` 까지 크기 지정이 가능하다
 - 앞에붙은 `"#"` 개수에 따라 바뀐다.

##### 예제
```
# 헤더 H1
## 헤더 H2
### 헤더 H3
#### 헤더 H4
##### 헤더 H5
###### 헤더 H6
```
##### 결과

# 헤더 H1
## 헤더 H2
### 헤더 H3
#### 헤더 H4
##### 헤더 H5
###### 헤더 H6

<br>
  
 - `H1`과 `H2`의 경우 각 `=====` 와 `-----`로 대체 가능하다  
  
##### 예제
```
헤더 H1
=====

헤더 H2
-----
```

##### 결과  

헤더 H1
===============

헤더 H2
---------------

# 2. 글자 강조 - Emphasis
 - `언더바(_)` 와 `별(*)` 를 이용하여 글씨를 강조할 수 있다.
 - html 문법을 통해서도 강조 할 수 있다.

##### 예제

```
이탤릭체 => *이탤릭체* or _이탤릭체_  
두껍게 => **두껍게** or __두껍게__  
두꺼운 이탤릭체 => **_두꺼운 이탤릭체_** or __*두꺼운 이탤릭체*__  

취소선 => ~~취소선~~
밑줄 => <u>밑줄</u>
```

##### 결과

이탤릭체 => *이탤릭체* or _이탤릭체_  
두껍게 => **두껍게** or __두껍게__  
두꺼운 이탤릭체 => **_두꺼운 이탤릭체_** or __*두꺼운 이탤릭체*__  

취소선 => ~~취소선~~  
밑줄 => <u>밑줄</u>

# 3. 들여쓰기 - Indent

 - 기본적으로 마크다운은 들여쓰기를 지원하지 않는다.
 - 공백 1칸 `&nbsp;` 공백 2칸 `&ensp;` 공백 4칸 `&emsp;`을 이용하자

##### 예제

```
&nbsp;안녕하세요
&ensp;안녕하세요
&emsp;안녕하세요
&nbsp;&nbsp;&nbsp;&nbsp;안녕하세요
```

##### 결과

&nbsp;안녕하세요  
&ensp;안녕하세요  
&emsp;안녕하세요  
&nbsp;&nbsp;&nbsp;&nbsp;안녕하세요


 - 예문 들여쓰기의 경우 탭(Tap) 또는 4칸 공백으로 사용할 수 있다.
 - 위아래가 빈줄이 아닐 경우 제대로 인식이 안될 수 있다.
   
##### 예제

```
안녕하세요  

    들여쓰기 메모

중입니다.
```

##### 결과

안녕하세요  

    들여쓰기 메모

중입니다.

# 4. 줄바꿈 - Line break
 - 라인 끝에 공백 2칸을 띄우면 줄바꿈이 된다.
 - `<br>`을 통해 강제 줄바꿈도 가능하다.

##### 예제

 ```
 **빈칸x**  
 안녕하세요
 메모중입니다.  

 **공백2칸**  
 안녕하세요  
 메모중입니다.

 **강제 줄 바꿈**  
 안녕하세요<br>메모중입니다.
 ```

##### 결과

 **빈칸x**  
 안녕하세요
 메모중입니다.  

 **2칸**  
 안녕하세요  
 메모중입니다.

 **강제 줄바꿈**  
 안녕하세요<br>메모중입니다.

# 5.  수평선 - Horizon
- `가로줄(-)`, `별(*)`, `밑줄(_)` 을 3개 이상 작성하면 수평선이 생긴다.
- `가로줄(-)`을 사용할 경우 헤더로 인식할 수 있으니 이전 줄을 비워야 한다.

##### 예제

```
가로줄

---
별

***

밑줄

___

가로줄  
---

```

##### 결과

가로줄  

---

별

***

밑줄 

___

가로줄  
---
=> H2 헤더로 인식된다.

# 6.  코드 블럭 - Code Block
 - ``백틱(`)`` 사이에 코드를 넣어 사용가능하다.
 - 3중 백틱 ```` ``` ```` 사이에 코드를 넣어 코드 블럭으로 사용 가능하다.
 - ``백틱(`)``으로 단어를 묶으면 강조의 효과도 볼 수 있다.
 - 백틱을 코드 블럭사이에 넣으려면 백틱을 하나 추가해 묶어주면 사용가능하다. ` => ``

##### 예제

````
```
코드 작성 구간
```
`강조`
````

##### 결과

```
코드 작성 구간
```
`강조`

# 7.  목록 - List


#### 순서있는 목록
 - 숫자+점(.)으로 사용한다.

##### 예제

 ```
 1. 하나
 2. 둘
 3. 셋
 ```

##### 결과

 1. 하나
 2. 둘
 3. 셋

 - 순서가 바뀌어도 내림차순으로 정렬된다.


##### 예제

 ```
 1. 하나
 3. 셋
 2. 둘
 ```

##### 결과

 1. 하나
 3. 셋
 2. 둘

#### 순서없는 목록
 - 글머리 기호(`*`,`+`,`-`)을 지원한다.
 - 혼합해서 사용 가능하다.

##### 예제

 ```
- 하나
    - 둘
        - 셋

* 하나
    + 둘
        - 셋
 ```

##### 결과

 - 하나
    - 둘
        - 셋

* 하나
    + 둘
        - 셋

# 9.  표 - Table

셀 정렬은 :기호(Colons)
 - 가로선(-) 3개이상으로 제목과 내용 구분한다.
 - 칸 구분은 수직선(\|)을 사용한다.
 - 정렬은 콜론(:) 기호의 위치로 정해진다.

##### 예제


```
|제목1|제목2|제목3|
|:---|---:|:---:|
|왼쪽|오른쪽|가운데|
|**강조**|*이탤릭*|__*이탤릭강조*__|
```

##### 결과

|제목1|제목2|제목3|
|:---|---:|:---:|
|왼쪽|오른쪽|가운데|
|**강조**|*이탤릭*|__*이탤릭강조*__|
